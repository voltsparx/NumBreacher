# NumBreacher

NumBreacher is a cybersecurity OSINT and telecom reconnaissance tool designed to analyze phone numbers safely and ethically.

## Features

- Country & region detection
- Telecom provider lookup
- VoIP detection
- Risk scoring
- OSINT link generator (Truecaller, Google, WhatsApp, Telegram)
- Bulk scanning
- JSON export
- CLI shell interface

## Installation
```
git clone https://github.com/voltsparx/NumBreacher.git
cd NumBreacher
pip install -r requirements.txt  
python numbreacher.py
```

## Usage

scan +919876543210  
bulk numbers.txt  
exportjson report.json
